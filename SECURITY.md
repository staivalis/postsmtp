# Security Policy

## Supported Versions

Security fixes are provided for the latest released version on PyPI.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub.

- Preferred (private): GitHub Security Advisories "Report a vulnerability"
  - https://github.com/staivalis/postsmtp/security/advisories
- Fallback (public): open a GitHub issue
  - https://github.com/staivalis/postsmtp/issues
- Include:
  - Affected version
  - Reproduction steps or proof of concept
  - Impact assessment

You should receive an acknowledgment within 72 hours.

## Disclosure Process

- We will validate the report and assess severity.
- We will prepare and publish a fix as soon as possible.
- We will coordinate public disclosure after a fix is available.
